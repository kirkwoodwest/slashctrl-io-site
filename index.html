/CTRL — Modular Bitwig Extension & Control System

A modular control system for Bitwig.
Built in the open. Shaped by use.

MIDI Hardware

Build configurations for any MIDI controller. Combine multiple controllers into a single system with shared state and cross-communication.

OSC

Structured, bidirectional OSC for every part of /CTRL. Designed for external tools, TouchOSC, and custom workflows.

Commands

A command-based architecture for control behavior. Compose actions, remap behavior live, and reconfigure without restarting or rebuilding.

Macros

Reusable behaviors that grow with your setup. Combine commands and mappings without rebuilding your Bitwig extension.

Recall

Capture and restore system state. Recall parameters, mappings, and behaviors consistently across sessions.

Components

Track Banks, Cursor Tracks, Devices, Remote Controls, Transport, Actions, Clip Launchers. All modular. All addressable. Remapped on the fly.

A long-term system, developed in public.
